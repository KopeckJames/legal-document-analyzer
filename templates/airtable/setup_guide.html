Airtable Setup Guide

Follow these steps to set up your Airtable integration
Step 1: Create an Airtable Account

If you don't already have an Airtable account, sign up at airtable.com.

Step 2: Generate an API Key
  1. Log in to your Airtable account
  2. Navigate to your account page
  3. Under the API section, click "Generate API key" if you don't already have one
  4. Copy your API key for use in the next step
Step 3: Connect to LegalDataInsights

Return to our application and enter your API key to connect your Airtable account.

Step 4: Configure Your Base

You can either:

  • Create a new Airtable base (recommended)
  • Connect to an existing base if you already have a suitable structure
Important: For security reasons, your Airtable API key provides complete access to all your Airtable bases. Keep it secure and never share it with others.
