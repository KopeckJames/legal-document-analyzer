Airtable Setup Guide

Follow these steps to set up your Airtable integration
Step 1: Create an Airtable Account

If you don't already have an Airtable account, sign up at airtable.com.

Step 2: Generate a Personal Access Token
  1. Log in to your Airtable account
  2. Navigate to your Personal Access Tokens page
  3. Click "Create new token"
  4. Name your token and select the necessary scopes (data.records:read, data.records:write, schema.bases:read, schema.bases:write)
  5. Copy your Personal Access Token for use in the next step
Step 3: Connect to LegalDataInsights

Return to our application and enter your Personal Access Token to connect your Airtable account.

Step 4: Configure Your Base

You can either:

  • Create a new Airtable base (recommended)
  • Connect to an existing base if you already have a suitable structure
Important: For security reasons, your Airtable Personal Access Token provides access to your Airtable bases according to the scopes you've granted. Keep it secure and never share it with others.
